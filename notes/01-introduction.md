## What is GitHub Copilot?

GitHub Copilot is an AI coding assistant developed by **GitHub** and powered by **OpenAI Codex**, a generative pretrained language model created by OpenAI.

It acts like an AI pair programmer that supports developers while coding.

---

## How GitHub Copilot Helps Developers

Research shows that when developers use GitHub Copilot:
- They can write code faster
- They focus more on solving bigger problems
- They stay in the coding flow for longer
- They feel more productive and fulfilled in their work

---

## Supported Editors and IDEs

GitHub Copilot is available as an extension for:
- Visual Studio Code (VS Code)
- Visual Studio
- Neovim
- JetBrains IDEs (IntelliJ IDEA, PyCharm, etc.)
